## continuous docs

Continuous documentation for your mission-critical software systems

We intelligently identify, extract, compile, and format engineering documentation in a huge variety of formats from your source tree, turning it into a single, up-to-date source of truth.

Features

No configuration

We generate full documentation for your project with no configuration. Just connect it to your repository and specify where the output should go, in what format.

Rich input format support

We support almost all the formats you already use, including code-based systems like JavaDoc and JsDoc, API systems including Swagger/OpenAPI, and Markdown and many other markup languages.

Confidentiality

We fully protect your IP. Not one byte of your codebase is ever stored on our servers.

Full semantic search

We provide lightning-fast search for the entire document set.

Analytics

Real-time analytics helps your users find the things most important to them, and alerts you to areas that could need improvement.
